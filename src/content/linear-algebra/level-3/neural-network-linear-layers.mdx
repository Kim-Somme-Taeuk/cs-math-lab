## 왜 배우나

신경망의 많은 층은 기본적으로 `y = Wx + b` 형태의 선형 계층에서 시작합니다. 입력 벡터를 다른 특징 공간으로 보내고, 비선형 활성화를 더해 복잡한 함수를 표현합니다.

핵심은 <strong>가중치 행렬을 입력 특징을 섞어 새로운 특징 표현을 만드는 변환으로 읽는 것</strong>입니다.

## 먼저 감 잡기

이미지, 문장, 사용자 정보는 벡터로 표현될 수 있습니다. 선형 계층은 이 벡터의 성분들을 가중합해 다음 층이 쓰기 좋은 특징으로 바꿉니다.

## 핵심 정의

### 입력 벡터

모델이 처리할 숫자 표현입니다. 픽셀, 임베딩, feature가 될 수 있습니다.

### 가중치 행렬

입력 특징을 어떤 비율로 섞어 출력 특징을 만들지 정하는 행렬입니다.

### bias

선형변환 뒤에 더하는 이동 성분입니다. 아핀 변환과 비슷하게 볼 수 있습니다.

### 배치 행렬곱

여러 입력을 한 번에 처리하기 위해 입력들을 행렬로 묶고 한 번에 곱합니다.

### 비선형 활성화

선형 계층만 여러 번 쌓으면 결국 하나의 선형변환처럼 합쳐집니다. 복잡한 패턴을 표현하려면 비선형성이 필요합니다.

## 자주 헷갈리는 지점

<strong>가중치를 단순한 점수표로만 보는 실수.</strong> 행렬은 입력 공간에서 출력 공간으로 가는 변환입니다.

<strong>bias를 무시하는 실수.</strong> bias는 출력 공간에서 기준점을 옮기는 역할을 합니다.

<strong>선형 계층만 많이 쌓으면 충분하다고 보는 실수.</strong> 비선형 활성화가 없으면 여러 선형변환은 하나로 합쳐질 수 있습니다.

<strong>배치 처리를 반복문으로만 생각하는 실수.</strong> 실제 구현은 큰 행렬 곱으로 병렬화됩니다.

## 코드로 읽기

```python
output = inputs @ weights.T + bias
```

배치 입력 `inputs`는 여러 샘플을 행렬로 묶은 것입니다. 한 번의 행렬 곱으로 모든 샘플의 출력 특징을 계산합니다.

```python
hidden = relu(output)
```

비선형 활성화가 들어가야 여러 층이 단순한 하나의 선형변환을 넘어설 수 있습니다.
