## 왜 배우나

선형변환은 벡터를 다른 벡터로 보내는 규칙입니다. 그래픽스에서 물체를 회전하거나 키우고, 게임에서 좌표를 바꾸고,
데이터를 다른 좌표계로 옮기는 감각과 연결됩니다.

이 장의 핵심은 <strong>행렬을 벡터를 움직이는 변환 규칙으로 읽는 법</strong>입니다.
핵, 상, 랭크-널리티 정리 같은 이론은 다루지 않습니다.

## 먼저 감 잡기

2D 벡터 `[1, 0]`을 90도 회전하면 `[0, 1]`이 됩니다. 어떤 규칙이 모든 벡터를 같은 방식으로 회전시킨다면
그 규칙을 변환으로 볼 수 있습니다.

행렬은 이런 변환을 숫자 표로 표현하는 방법입니다.

## 핵심 정의

### 벡터를 다른 벡터로 보내는 규칙

변환은 입력 벡터를 받아 출력 벡터를 만듭니다.

```text
T([x, y]) = [2x, y]
```

이 예시는 x방향으로 두 배 늘리는 변환입니다.

### 회전, 스케일, 반사, 투영 맛보기

- 회전: 방향을 돌립니다.
- 스케일: 크기를 키우거나 줄입니다.
- 반사: 거울처럼 뒤집습니다.
- 투영: 어떤 방향의 정보만 남깁니다.

### 행렬과 선형변환

많은 선형변환은 행렬로 표현할 수 있습니다. 그래서 행렬-벡터 곱은 변환을 적용하는 계산처럼 읽을 수 있습니다.

## 자주 헷갈리는 지점

<strong>모든 함수가 선형변환이라고 생각하는 실수.</strong> 여기서는 벡터 구조를 보존하는 변환만 선형변환으로 봅니다.

<strong>행렬과 변환을 완전히 다른 것으로 보는 실수.</strong> 행렬은 선형변환을 표현하는 방법입니다.

<strong>투영을 되돌릴 수 있다고 생각하는 실수.</strong> 한 방향 정보를 없애면 원래 값을 완전히 복원하기 어렵습니다.

<strong>변환 순서를 무시하는 실수.</strong> 회전한 뒤 스케일하는 것과 스케일한 뒤 회전하는 것은 다를 수 있습니다.

## 코드로 읽기

```python
def transform(A, v):
    return [
        sum(A[row][col] * v[col] for col in range(len(v)))
        for row in range(len(A))
    ]
```

그래픽스에서는 좌표 벡터에 변환 행렬을 곱해 새 좌표를 만듭니다.

```python
new_position = transform_matrix @ position
```
