## 왜 배우나

벡터 연산은 위치를 이동시키고, 방향과 크기를 바꾸고, 여러 특징을 섞는 기본 방법입니다. 코드에서는 리스트의 같은 위치끼리
더하거나, 모든 성분에 같은 수를 곱하는 배열 연산으로 자주 나타납니다.

이 장의 핵심은 <strong>벡터 덧셈과 스칼라 곱을 좌표 이동과 가중합 감각으로 읽는 법</strong>입니다.
span의 엄밀한 정의는 다루지 않습니다.

## 먼저 감 잡기

캐릭터 위치가 `[3, 2]`이고 속도가 `[1, -1]`이면 다음 위치는 `[4, 1]`입니다.

```text
[3, 2] + [1, -1] = [4, 1]
```

방향 벡터 `[3, 2]`에 2를 곱하면 `[6, 4]`가 됩니다. 방향은 같고 크기만 커진 것으로 볼 수 있습니다.

## 핵심 정의

### 벡터 덧셈

벡터 덧셈은 같은 위치의 성분끼리 더합니다.

```text
[a, b] + [c, d] = [a + c, b + d]
```

2D에서는 한 이동 뒤에 또 다른 이동을 이어 붙이는 그림으로 볼 수 있습니다.

### 스칼라 곱

스칼라는 벡터 전체에 곱하는 숫자입니다.

```text
2[3, 2] = [6, 4]
```

양수는 크기를 바꾸고, 음수는 방향을 뒤집는 감각을 줍니다.

### 선형결합 맛보기

벡터에 스칼라를 곱한 뒤 더하면 선형결합입니다.

```text
2u + 3v
```

입문 단계에서는 "여러 벡터를 가중치로 섞는 것" 정도로 잡으면 됩니다.

## 자주 헷갈리는 지점

<strong>성분 개수가 다른 벡터를 더하려는 실수.</strong> `[1, 2]`와 `[1, 2, 3]`은 바로 더할 수 없습니다.

<strong>스칼라 곱을 한 성분에만 적용하는 실수.</strong> 스칼라는 모든 성분에 곱합니다.

<strong>음수 스칼라의 의미를 놓치는 실수.</strong> 음수를 곱하면 방향이 반대로 뒤집히는 감각이 생깁니다.

<strong>선형결합을 어려운 공식으로만 보는 실수.</strong> 여러 벡터를 가중치로 섞는다고 보면 됩니다.

## 코드로 읽기

```python
def add(u, v):
    return [u[i] + v[i] for i in range(len(u))]

def scale_vector(c, v):
    return [c * value for value in v]
```

NumPy를 쓰면 배열 연산으로 더 간단히 표현합니다.

```python
next_position = position + velocity
weighted = 0.7 * image_a + 0.3 * image_b
```
