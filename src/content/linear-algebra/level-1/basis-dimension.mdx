## 왜 배우나

기저와 차원은 좌표가 어떤 기준으로 표현되는지 이해하게 해 줍니다. 같은 점도 기준 벡터를 어떻게 잡느냐에 따라
좌표 표현이 달라질 수 있습니다.

이 장의 핵심은 <strong>기저를 공간을 만드는 기준 벡터, 차원을 필요한 독립 방향의 수로 읽는 법</strong>입니다.
선형독립의 엄밀한 증명은 다루지 않습니다.

## 먼저 감 잡기

2D 평면에서는 보통 `[1, 0]`과 `[0, 1]`을 기준 방향으로 씁니다. `[3, 2]`는 `[1, 0]`을 3번, `[0, 1]`을 2번
쓴 좌표라고 볼 수 있습니다.

```text
[3, 2] = 3[1, 0] + 2[0, 1]
```

## 핵심 정의

### 기저

기저는 공간의 모든 벡터를 만들 수 있는 기준 벡터들의 묶음입니다.

입문 단계에서는 "좌표를 만드는 기준 방향"으로 이해하면 됩니다.

### 차원

차원은 공간을 표현하는 데 필요한 독립적인 방향의 수입니다. 2D 평면은 두 방향이 필요하고, 3D 공간은 세 방향이 필요합니다.

### 표준기저

2D 표준기저는 다음 두 벡터입니다.

```text
e1 = [1, 0]
e2 = [0, 1]
```

### 좌표 표현

좌표는 기준 벡터를 얼마나 섞을지 나타내는 숫자입니다.

```text
[3, 2] = 3e1 + 2e2
```

## 자주 헷갈리는 지점

<strong>차원을 화면의 축 개수로만 보는 실수.</strong> 데이터에서는 특징 수가 차원이 될 수 있습니다.

<strong>기저를 하나의 벡터라고 생각하는 실수.</strong> 보통 기저는 여러 기준 벡터의 묶음입니다.

<strong>기준이 바뀌어도 좌표가 항상 같다고 보는 실수.</strong> 같은 벡터도 기준이 바뀌면 좌표 숫자가 달라질 수 있습니다.

<strong>선형독립 증명부터 파고드는 실수.</strong> 여기서는 필요한 독립 방향의 수라는 감각이 먼저입니다.

## 코드로 읽기

```python
e1 = [1, 0]
e2 = [0, 1]
v = [3, 2]
```

`v = [3, 2]`는 표준기저 기준으로 `3 * e1 + 2 * e2`라고 읽을 수 있습니다.

데이터에서는 특징이 128개라면 128차원 벡터로 다루는 식입니다.
