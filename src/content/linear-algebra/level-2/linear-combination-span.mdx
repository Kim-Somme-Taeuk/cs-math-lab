## 왜 배우나

선형결합은 벡터를 스칼라배해서 더하는 구조입니다. span은 그렇게 만들 수 있는 모든 벡터의 모음입니다.
컴공에서는 특징 벡터를 섞거나, 어떤 좌표가 주어진 기준 벡터들로 표현 가능한지 판단할 때 이 감각이 필요합니다.

핵심은 <strong>벡터들이 만들 수 있는 방향과 영역을 읽는 법</strong>입니다. 일반 벡터공간 공리는 다루지 않습니다.

## 먼저 감 잡기

벡터 u와 v가 있으면 `2u - 3v`처럼 스칼라를 곱하고 더할 수 있습니다. u 하나만 있으면 그 벡터가 놓인 직선 위만 만들 수 있고,
서로 다른 두 방향 u, v가 있으면 2D 평면 전체를 만들 수도 있습니다.

## 핵심 정의

### 선형결합

선형결합은 벡터들에 숫자를 곱한 뒤 더한 것입니다.

```text
a u + b v
```

여기서 a와 b는 스칼라입니다.

### span

span은 주어진 벡터들로 만들 수 있는 모든 선형결합의 집합입니다.

```text
span(u, v) = { a u + b v | a, b는 숫자 }
```

한 벡터의 span은 보통 직선 감각이고, 서로 다른 두 방향의 span은 평면 감각으로 볼 수 있습니다.

### 좌표 표현

`[3, 2] = 3[1, 0] + 2[0, 1]`처럼 좌표는 기준 벡터의 선형결합 계수로 읽을 수 있습니다.

## 자주 헷갈리는 지점

<strong>span을 벡터 하나로 착각하는 실수.</strong> span은 벡터 하나가 아니라 만들 수 있는 모든 결과의 모음입니다.

<strong>스칼라를 한 번만 고정하는 실수.</strong> span에서는 가능한 모든 스칼라 값을 생각합니다.

<strong>방향이 같은 두 벡터가 평면을 만든다고 보는 실수.</strong> 같은 직선 위 벡터들은 새 방향을 만들지 못합니다.

<strong>좌표를 그냥 숫자 묶음으로만 보는 실수.</strong> 좌표는 기준 벡터를 얼마나 섞는지 나타내는 값입니다.

## 코드로 읽기

```python
def combine(a, u, b, v):
    return [a * u[i] + b * v[i] for i in range(len(u))]
```

추천 점수, 임베딩 평균, 이미지 블렌딩처럼 여러 벡터를 가중치로 섞는 계산은 선형결합 감각과 연결됩니다.
