## 왜 배우나

행렬식은 행렬 변환이 2D 면적이나 3D 부피를 얼마나 스케일하는지 알려 줍니다. 값이 0이면 공간이 납작해져 정보가
사라지고, 역행렬이 없다는 감각과 연결됩니다.

핵심은 <strong>행렬식으로 변환이 공간을 보존하는지, 눌러 없애는지 판단하는 법</strong>입니다.
큰 행렬 행렬식 계산 훈련은 다루지 않습니다.

## 먼저 감 잡기

2D에서 정사각형을 행렬로 변환하면 평행사변형이 됩니다. 행렬식의 절댓값은 그 면적이 몇 배가 되었는지 알려 줍니다.

행렬식이 0이면 면적이 0이 됩니다. 평면이 선으로 눌린 것처럼 정보가 사라졌다고 볼 수 있습니다.

## 핵심 정의

### 면적/부피 스케일

2D에서는 행렬식의 절댓값이 면적 스케일, 3D에서는 부피 스케일 감각과 연결됩니다.

### 0이면 정보가 사라짐

행렬식이 0이면 어떤 방향이 눌려 면적이나 부피가 사라진 것입니다. 서로 다른 입력이 같은 출력으로 갈 수 있습니다.

### 역행렬 존재 여부

정사각 행렬에서 행렬식이 0이 아니면 역행렬이 존재하는 감각과 연결됩니다. 0이면 되돌리기 어렵습니다.

### 방향 뒤집힘

행렬식 부호는 방향이 뒤집혔는지 맛보기로 알려 줍니다. 절댓값은 크기 변화, 부호는 방향 감각입니다.

## 자주 헷갈리는 지점

<strong>행렬식을 단순 계산 공식으로만 보는 실수.</strong> 핵심은 공간 스케일과 정보 손실 해석입니다.

<strong>행렬식 0을 작은 변화라고 생각하는 실수.</strong> 0은 어떤 방향이 완전히 눌렸다는 강한 신호입니다.

<strong>부호와 크기를 섞는 실수.</strong> 절댓값은 면적/부피 배율, 부호는 방향 뒤집힘 감각입니다.

<strong>큰 행렬 계산을 먼저 외우려는 실수.</strong> 여기서는 작은 예시로 해석을 잡습니다.

## 코드로 읽기

```python
def det2(A):
    return A[0][0] * A[1][1] - A[0][1] * A[1][0]
```

실무에서는 직접 공식보다 라이브러리 함수를 쓰지만, 결과를 해석하는 감각이 중요합니다.

```python
if abs(det) < 1e-9:
    print("되돌리기 어려운 변환일 수 있음")
```
