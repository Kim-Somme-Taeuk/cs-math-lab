## 왜 배우나

변화율은 입력이 바뀔 때 출력이 얼마나 바뀌는지 보는 언어입니다. 속도, 성능 그래프, 손실 함수 변화, 프레임별 움직임을 읽으려면 변화율 감각이 필요합니다.

이 장의 핵심은 <strong>공식을 외우기 전에 변화량과 기울기를 해석하는 법</strong>입니다.

## 먼저 감 잡기

입력 변화량은 `나중 입력 - 처음 입력`이고, 출력 변화량은 `나중 출력 - 처음 출력`입니다. 평균 변화율은 출력 변화량을 입력 변화량으로 나눈 값입니다.

```text
평균 변화율 = 출력 변화량 / 입력 변화량
```

## 핵심 정의

평균 변화율은 두 입력 사이 전체 구간의 변화 속도입니다. 값이 양수면 출력이 증가했고, 음수면 감소했습니다. 절댓값이 크면 같은 입력 변화에 출력이 더 많이 변한 것입니다.

## 자주 헷갈리는 지점

<strong>변화량과 변화율을 섞는 실수.</strong> 변화량은 차이이고, 변화율은 그 차이를 입력 변화로 나눈 비율입니다.

<strong>기울기가 항상 시간 속도라고 보는 실수.</strong> 입력이 시간이면 속도지만, 입력이 데이터 크기면 성능 변화율이 됩니다.

<strong>평균 변화율을 순간 변화율로 착각하는 실수.</strong> 평균 변화율은 구간 전체를 한 번에 본 값입니다.

## 직접 실험하기

두 점 사이 간격을 줄이며 평균 변화율이 한 점 근처 기울기와 가까워지는 모습을 확인합니다.

## 코드로 읽기

```python
before_size = 100
after_size = 200
before_time = 0.03
after_time = 0.09

rate = (after_time - before_time) / (after_size - before_size)
```

이 코드는 입력 크기가 늘 때 실행 시간이 얼마나 빨리 증가하는지 대략 읽는 예시입니다.

## 전공에서 만나는 장면

- 시간에 따른 위치 변화
- 입력 크기에 따른 실행 시간 증가
- 학습률 변화에 따른 손실 변화
- 사용자 수 증가에 따른 서버 비용 변화
