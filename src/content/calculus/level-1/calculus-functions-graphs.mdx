## 왜 배우나

미적분에서 함수는 변화율을 읽기 위한 출발점입니다. 컴퓨터공학에서는 함수가 데이터 변환, 그래픽스 좌표 변환, 모델의 입력과 출력처럼 계속 등장합니다.

이 장의 핵심은 <strong>함수를 공식 암기 대상이 아니라 입력을 출력으로 보내는 규칙으로 읽는 법</strong>입니다. 역함수와 합성함수 심화는 뒤로 미룹니다.

## 먼저 감 잡기

함수는 입력 하나를 받으면 정해진 출력 하나를 돌려주는 규칙입니다. 예를 들어 `f(x) = 2x + 1`은 입력 `x`를 두 배로 키우고 1을 더합니다.

그래프는 입력과 출력의 관계를 그림으로 펼친 것입니다. x축은 입력, y축은 출력으로 보면 됩니다. 선형 함수는 일정하게 늘고, 이차 함수는 휘어지며, 지수 함수는 점점 더 빠르게 커집니다.

## 핵심 정의

정의역은 넣을 수 있는 입력의 범위입니다. 공역은 출력이 들어갈 수 있다고 정한 전체 범위이고, 치역은 실제로 나오는 출력들의 모음입니다.

```text
f(x) = x + 2
입력 3 -> 출력 5
```

코드의 함수는 수학 함수와 비슷하지만 완전히 같지는 않습니다. 코드 함수는 파일을 읽거나 화면을 바꾸는 부수 효과를 만들 수 있지만, 수학 함수는 같은 입력에 같은 출력을 주는 규칙으로 먼저 봅니다.

## 자주 헷갈리는 지점

<strong>함수를 코드 문법으로만 보는 실수.</strong> 수학에서는 입력과 출력의 관계가 먼저입니다.

<strong>그래프를 그림 모양 암기로 보는 실수.</strong> 그래프는 x가 바뀔 때 y가 어떻게 바뀌는지 보여 줍니다.

<strong>공역과 치역을 섞는 실수.</strong> 공역은 미리 정한 출력 후보 전체, 치역은 실제 출력입니다.

<strong>입력 하나에 출력 여러 개가 나와도 함수라고 보는 실수.</strong> 입문 단계에서는 입력 하나에 출력 하나라는 규칙을 먼저 잡습니다.

## 코드로 읽기

```python
def score_to_grade(score):
    if score >= 90:
        return "A"
    if score >= 80:
        return "B"
    return "C"
```

이 코드는 점수를 등급으로 보내는 규칙입니다. 수학 함수처럼 읽으려면 먼저 입력이 무엇이고 출력이 무엇인지 정해야 합니다.

## 전공에서 만나는 장면

- 점수를 등급으로 바꾸는 규칙
- 픽셀 좌표를 화면 위치로 바꾸는 규칙
- 입력 데이터에서 예측값을 만드는 모델
- 시간 값을 애니메이션 위치로 바꾸는 함수
