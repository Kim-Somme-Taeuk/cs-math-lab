## 왜 배우나

도함수는 그래프가 어디서 증가하고 감소하는지 알려 주는 신호입니다. 데이터를 볼 때도 값 자체뿐 아니라 변하는 방향과 속도가 중요합니다.

## 먼저 감 잡기

도함수가 양수면 함수는 보통 증가하고, 음수면 감소합니다. 도함수의 크기가 크면 더 가파르게 변합니다. 도함수가 0인 지점은 꼭대기나 바닥 후보가 될 수 있습니다.

## 핵심 정의

도함수의 부호는 증가와 감소를 읽게 해 줍니다. 도함수의 크기는 변화 속도를 나타냅니다. 도함수가 0인 지점은 평평한 지점이며, 극대나 극소 후보가 될 수 있습니다.

## 자주 헷갈리는 지점

<strong>도함수 0이면 무조건 최댓값이라고 보는 실수.</strong> 후보일 뿐이고 주변 변화를 봐야 합니다.

<strong>함수값과 도함수값을 섞는 실수.</strong> 높은 위치라도 기울기는 0일 수 있습니다.

<strong>부호보다 공식 계산만 보는 실수.</strong> 그래프 해석에서는 양수, 음수, 0의 의미가 중요합니다.

## 코드로 읽기

```python
def trend(derivative_value):
    if derivative_value > 0:
        return "increasing"
    if derivative_value < 0:
        return "decreasing"
    return "flat"
```

기울기 값을 신호처럼 읽으면 그래프의 움직임을 분류할 수 있습니다.
