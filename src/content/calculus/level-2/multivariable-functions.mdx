## 왜 배우나

실제 컴공 문제에서는 입력이 하나뿐인 경우보다 여러 개인 경우가 많습니다. 모델의 손실은 여러 파라미터에 의해 바뀌고, 이미지는 x좌표와 y좌표로 픽셀을 고르며, 게임 위치도 여러 좌표로 표현됩니다.

이 장의 핵심은 <strong>여러 입력이 하나의 출력에 함께 영향을 주는 함수를 읽는 법</strong>입니다. 다변수 함수의 엄밀한 위상적 정의는 다루지 않습니다.

## 먼저 감 잡기

`f(x, y)`는 입력이 두 개인 함수입니다. 예를 들어 `f(x, y) = x^2 + y^2`라면 x와 y가 모두 출력값에 영향을 줍니다.

그래프를 3D 지형처럼 떠올릴 수 있고, 같은 높이를 연결하면 등고선처럼 볼 수도 있습니다.

## 핵심 정의

다변수 함수는 여러 입력을 받아 하나 이상의 출력을 만듭니다.

```text
f(x, y) = x^2 + y^2
입력: x, y
출력: 하나의 값
```

등고선은 같은 출력값을 만드는 입력들의 모음입니다. 손실 함수에서는 같은 손실값을 만드는 파라미터 조합을 상상하는 데 도움이 됩니다.

## 자주 헷갈리는 지점

<strong>입력이 두 개면 출력도 반드시 두 개라고 보는 실수.</strong> 입력이 여러 개여도 출력은 하나일 수 있습니다.

<strong>3D 그림을 못 그리면 이해할 수 없다고 보는 실수.</strong> 표, 등고선, 코드 함수로도 충분히 읽을 수 있습니다.

<strong>각 입력의 영향이 항상 독립이라고 보는 실수.</strong> `xy` 같은 항은 두 입력이 함께 출력에 영향을 주는 구조입니다.

## 직접 판별하기

아래 함수에서 입력과 출력을 찾아봅니다.

- 이미지 밝기 `brightness(x, y)`
- 손실 함수 `loss(weight, bias)`
- 게임 높이맵 `height(x, z)`

## 코드로 읽기

```python
def loss(weight, bias):
    prediction = weight * x + bias
    return (prediction - target) ** 2
```

이 함수는 `weight`와 `bias`라는 두 입력이 손실값 하나에 영향을 줍니다. 어느 입력을 바꿨는지에 따라 손실이 다르게 움직입니다.

## 전공에서 만나는 장면

- 머신러닝 손실 함수
- 이미지의 x, y 픽셀 좌표
- 게임 월드의 위치와 높이
- 물리 시뮬레이션의 여러 상태 변수
