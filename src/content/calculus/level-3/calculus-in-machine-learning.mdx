## 왜 배우나

머신러닝에서 미적분은 따로 떨어진 공식 묶음이 아니라 학습 루프의 언어입니다. 모델이 예측을 만들고, 손실 함수가 오차를 숫자로 만들며, 그래디언트가 파라미터를 어느 방향으로 바꿀지 알려 줍니다.

이 장의 핵심은 <strong>모델 출력, 손실, 그래디언트, 파라미터 업데이트를 하나의 흐름으로 연결하는 법</strong>입니다. 신경망 전체 역전파 유도는 다루지 않습니다.

## 먼저 감 잡기

선형대수에서는 `y = Wx + b`가 입력을 출력으로 바꾸는 모델 구조를 줍니다. 미적분에서는 이 출력이 손실을 얼마나 바꾸는지, 파라미터를 어떻게 움직이면 손실이 줄어드는지를 봅니다.

학습은 손실을 줄이는 방향으로 파라미터를 조금씩 움직이는 과정입니다.

## 핵심 정의

머신러닝 학습 흐름은 다음처럼 볼 수 있습니다.

```text
입력 -> 모델 출력 -> 손실 함수 -> 그래디언트 -> 파라미터 업데이트
```

역전파는 연쇄법칙을 효율적으로 적용해 복잡한 모델의 그래디언트를 구하는 방법입니다.

## 자주 헷갈리는 지점

<strong>모델 구조와 학습 규칙을 섞는 실수.</strong> `y = Wx + b`는 출력 계산이고, 그래디언트 업데이트는 파라미터를 바꾸는 규칙입니다.

<strong>손실이 작아지는 방향을 감으로만 찾는 실수.</strong> 그래디언트가 방향 정보를 제공합니다.

<strong>역전파 전체 유도를 지금 다 알아야 한다고 보는 실수.</strong> 여기서는 연쇄법칙이 효율적으로 쓰인다는 감각만 잡습니다.

## 직접 판별하기

아래 흐름에서 각 단계의 역할을 말해 봅니다.

```text
x -> y_pred = Wx + b -> loss(y_pred, y_true) -> grad -> W, b update
```

## 코드로 읽기

```python
y_pred = model(x)
loss = mse(y_pred, y_true)
loss.backward()
optimizer.step()
```

`model`은 예측을 만들고, `loss`는 오차를 숫자로 만들며, `backward`는 그래디언트를 계산합니다. `step`은 그 그래디언트를 사용해 파라미터를 갱신합니다.

## 전공에서 만나는 장면

- 선형 회귀 학습
- 신경망 학습 루프
- 손실 곡선 모니터링
- 역전파와 자동미분 프레임워크
