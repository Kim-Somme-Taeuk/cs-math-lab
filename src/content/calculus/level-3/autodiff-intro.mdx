## 왜 배우나

현대 딥러닝 코드는 복잡한 합성함수의 미분을 사람이 손으로 전부 계산하지 않습니다. 자동미분은 계산 과정을 그래프로 추적하고, 연쇄법칙을 이용해 필요한 그래디언트를 계산합니다.

이 장의 핵심은 <strong>자동미분을 마법이 아니라 계산 그래프와 연쇄법칙의 조합으로 보는 법</strong>입니다. reverse-mode autodiff 전체 구현은 다루지 않습니다.

## 먼저 감 잡기

forward pass는 입력에서 출력과 손실을 계산하는 과정입니다. 그 과정에서 어떤 중간값이 어떤 연산으로 만들어졌는지 기록하면, 나중에 변화율을 뒤로 전달할 수 있습니다.

연쇄법칙은 각 단계의 변화율을 연결합니다. 자동미분은 이 연결을 코드가 대신 추적하는 감각입니다.

## 핵심 정의

계산 그래프는 입력, 중간값, 출력이 어떤 연산으로 이어지는지 나타냅니다.

```text
x -> multiply -> add -> loss
```

자동미분은 이 그래프에서 각 노드의 변화율을 추적해 최종 손실이 입력이나 파라미터에 얼마나 민감한지 계산합니다.

## 자주 헷갈리는 지점

<strong>자동미분을 수치미분과 같은 것으로 보는 실수.</strong> 자동미분은 작은 h로 근사하는 방식이 아니라 연산 규칙과 연쇄법칙을 추적합니다.

<strong>forward pass만 있으면 학습이 끝난다고 보는 실수.</strong> 학습에는 손실의 그래디언트가 필요합니다.

<strong>역전파 전체 구현을 지금 다 알아야 한다고 생각하는 실수.</strong> 여기서는 계산 그래프와 연쇄법칙 연결만 잡습니다.

## 직접 판별하기

다음 계산에서 중간값을 표시해 봅니다.

```text
z = w * x + b
loss = (z - target)^2
```

## 코드로 읽기

```python
z = w * x + b
loss = (z - target) ** 2
loss.backward()
```

실제 프레임워크에서는 `backward()` 같은 호출이 계산 그래프를 따라 그래디언트를 채웁니다. 핵심은 손실에서 파라미터까지 변화율을 거슬러 추적한다는 점입니다.

## 전공에서 만나는 장면

- PyTorch, TensorFlow의 gradient 계산
- 신경망 역전파
- 손실 함수와 파라미터 업데이트 연결
- 복잡한 모델의 미분 자동화
