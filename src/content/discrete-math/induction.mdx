## 1. 이 개념이 필요한 이유

수학적 귀납법은 무한히 많은 경우를 한 번에 다루는 증명 방법입니다. 재귀 함수와 반복문이
왜 모든 입력 크기에서 맞는지 설명할 때 자주 등장합니다.

## 2. 직관적인 설명

첫 번째 계단에 설 수 있고, 어느 계단에 설 수 있으면 다음 계단으로 갈 수 있다면 모든
계단에 도달할 수 있습니다.

## 3. 정확한 정의

명제 $P(n)$에 대해 $P(1)$이 참이고, 임의의 k에 대해 $P(k) \to P(k+1)$이 참이면 모든
자연수 n에 대해 $P(n)$이 참입니다.

## 4. 헷갈리는 부분

기저 사례와 귀납 단계 중 하나라도 빠지면 전체 주장을 보장할 수 없습니다.

## 7. 코드 연결

```ts
function sum(n: number): number {
  if (n === 1) return 1;
  return n + sum(n - 1);
}
```
