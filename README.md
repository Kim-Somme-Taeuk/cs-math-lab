# CS Math Lab

CS Math Lab은 컴퓨터공학 전공자가 전공 학습에 필요한 수학 기초를 단계적으로 익힐 수
있도록 돕는 인터랙티브 웹사이트입니다. 전체 구조는 이산수학, 선형대수, 미적분, 확률통계
대주제로 나누고, 현재 MVP는 이산수학 Level 1 입문 범위에 집중합니다.

## 기능

- 홈 페이지와 컴공 수학 로드맵
- MDX 기반 챕터 콘텐츠
- KaTeX 수식 렌더링
- 논리 조건식 빌더
- 조건문 위반 탐지기
- 집합 필터 실험실 및 벤 다이어그램
- 함수 매핑 직접 조작기
- 관계 성질 판별기
- 귀납법 증명 카드 조립기
- 경우의 수 시뮬레이터
- DFS/BFS 단계 실행기
- 챕터별 퀴즈와 즉시 피드백
- 챕터 끝 다음 학습 이동 링크
- 반응형 UI

## 로드맵 범위

최상위 대주제:

- 이산수학
- 선형대수
- 미적분
- 확률통계

각 대주제 안에는 Level이 있고, 각 Level 안에 챕터가 있습니다. 현재 상세 페이지가 열리는
활성 범위는 이산수학입니다.

이산수학의 현재 8개 챕터는 `Level 1. 입문`입니다. 전체 이산수학 커리큘럼이 아니라
전공 학습에 필요한 최소 입문 범위를 먼저 다루는 MVP입니다.

이산수학 Level 1 ready 콘텐츠:

- 명제와 논리
- 조건문과 진리표
- 집합과 집합 연산
- 함수
- 관계
- 수학적 귀납법
- 경우의 수
- 그래프 기초

이산수학 향후 확장 예정 레벨:

- `Level 2. 핵심 확장`: 증명 기법, 술어 논리, 동치관계, 점화식, 포함배제 원리 등
- `Level 3. 컴공 응용`: 알고리즘 분석, 이산확률, 정수론, 부울 대수, DAG와 위상 정렬 등

선형대수, 미적분, 확률통계는 각각 Level 1/2/3 구조와 planned 챕터 목록을 로드맵에 표시하고,
세부 챕터 페이지는 아직 만들지 않습니다.

예정 챕터는 로드맵에서 예정으로 표시하고, 일반 챕터 이동과 상세 페이지 접근에서는 ready
챕터만 노출합니다.

## 콘텐츠 원칙

- 기본 코드 예시는 Python을 먼저 사용합니다.
- TypeScript/JavaScript 예시는 필요한 경우에만 보조로 둡니다.
- 코드 전에는 자연어 상황과 수학적 분해를 먼저 제시합니다.
- 코드 후에는 수학 개념과 코드 표현의 대응 관계를 짧게 설명합니다.
- 코드 예시는 문법 학습이 아니라 수학 개념 연결을 위한 보조 자료로 사용합니다.

## 레이아웃 원칙

학습 본문, 인터랙티브 컴포넌트, 퀴즈의 여백과 모바일 동작 기준은
[docs/layout-guidelines.md](docs/layout-guidelines.md)를 따릅니다. 인터랙티브 섹션은 모바일
가로 밀림을 만들지 않고, 상태 변경 시 섹션 높이가 흔들리지 않도록 설계합니다.

## 실행 방법

```bash
npm install
npm run dev
```

`dev` 스크립트는 개발 서버 출력 폴더를 `.next-dev`로 분리해 실행합니다. 일반 Next.js 기본
동작 그대로 확인해야 할 때는 `dev:default`를 사용합니다.

```bash
npm run dev:default
```

이 작업환경에서는 `plana-ai`가 `localhost:3000`을 사용하므로, CS Math Lab을 확인할 때는
다음처럼 3001 포트로 실행합니다.

```bash
npm run dev -- --hostname 0.0.0.0 --port 3001
```

브라우저에서 `http://localhost:3001`로 접속합니다.

주의: `localhost:3001` 접근은 앱 코드가 아니라 ccc/Docker 포트 포워딩 설정에 의존합니다.
앱 서버가 컨테이너 내부에서 응답해도 호스트 포워딩이 없으면 브라우저의 `localhost:3001`은
열리지 않을 수 있습니다.
