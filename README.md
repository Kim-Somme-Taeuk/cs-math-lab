# CS Math Lab

CS Math Lab은 컴퓨터공학 전공자가 전공 학습에 필요한 수학 기초를 단계적으로 익힐 수
있도록 돕는 인터랙티브 웹사이트입니다. 초기 MVP는 이산수학 입문에 집중합니다.

## 기능

- 홈 페이지와 이산수학 로드맵
- MDX 기반 챕터 콘텐츠
- KaTeX 수식 렌더링
- 진리표 인터랙티브
- 집합 연산 및 벤 다이어그램 인터랙티브
- 반응형 UI

## MVP 범위

완성 콘텐츠:

- 명제와 논리
- 조건문과 진리표
- 집합과 집합 연산

확장용 초안:

- 함수
- 관계
- 수학적 귀납법
- 경우의 수
- 그래프 기초

## 기술 스택

- Next.js
- React
- TypeScript
- Tailwind CSS
- MDX
- KaTeX

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속합니다.

## 빌드

```bash
npm run build
```

## 향후 계획

- 함수와 관계 매핑 시각화
- 귀납법 도미노/계단 시각화
- 순열과 조합 시뮬레이터
- DFS/BFS 그래프 탐색 시각화
- 로컬 저장 기반 챕터 완료 체크
- 사이트 내 검색
